Junior Full Stack Project

for learning and practicing purposes

Technologies used:

ANGULAR
Injectable Singleton Service - to pass data between components through subscriptions
Routing - RouterModule configured
Renderer2 - to manipulate DOM elements
UI Design Features - SVGs and CSS animations
REST Access Service - with HttpClientModule and custom request headers (JWT)
JAVA SPRING BOOT
REST APIs - to access backend features
Spring Framework - dependency injection
Spring Security - including JWT authentication/authorization service
Spring Data JPA - connecting to the database
MySQL
User account data and game results in the database
Docker
Creating the image and deploying the application
